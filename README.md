# Marvel React App

## Quickstart

Install required dependencies:
```
yarn
```

Run development server:
```
npm run dev
```

Run prod:
```
npm run prod
```

Browser:
```
open http://localhost:3000/
```

Deploy prod:
```
npm run build
```

## Technologies

* ES6
* React
* Redux
* Webpack
* React Dev Tools
* Axios

<kbd>CTRL</kbd> + <kbd>H</kbd>
